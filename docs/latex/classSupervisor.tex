\doxysubsection{Detailed Description}
A class to handle the supervisor. 

This class allows updating the mission of the boat according to the section \char`\"{}\+MISSION CONFIG\char`\"{} in the file \mbox{\hyperlink{Config_8h}{Config.\+h}}. The mission is divided into two parts\+:
\begin{DoxyItemize}
\item the first part is to reach the next waypoint
\item the second part is to perform a task from this waypoint
\end{DoxyItemize}

To perform these tasks, the supervisor can select two kinds of algorithms\+: type 1 algorithms are used to perform the first part of the mission while type 2 algorithms are used to perform the second part. You can change the algorithms used for the mission, the waypoints to reach, and the duration of the type 2 algorithm in the file \mbox{\hyperlink{Config_8h}{Config.\+h}}.

{\footnotesize\ttfamily void Supervisor\+::update\+Mission (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Update the mission. 

This function updates the mission first by checking the status of the RC receiver to determine whether the boat should be autonomous or remotely controlled. Then, it checks if the boat has reached the current waypoint. If it has, it either switches to the next waypoint or tells the controller to use the type 2 algorithm for the specified duration, before switching to the next waypoint and reusing the type 1 algorithm.
