\doxysubsection{Detailed Description}
A class to handle the controller. 

This class contain functions to initialize and update the servo motors based on the data from the algorithm used for the mission. The algorithm to use is defined in \mbox{\hyperlink{Config_8h_source}{Config.\+h}}.
