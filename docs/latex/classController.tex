\doxysubsection{Detailed Description}
A class to handle the controller. 

This class contains functions to initialize and update the servo motors based on the data from the algorithmz used for the mission. The algorithms to use are defined in \mbox{\hyperlink{Config_8h_source}{Config.\+h}}.
