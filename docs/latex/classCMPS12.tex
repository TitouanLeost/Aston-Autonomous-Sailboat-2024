\doxysubsection{Detailed Description}
A class to handle the \mbox{\hyperlink{classCMPS12}{CMPS12}}. 

This class allows to get the yaw (filtered or not), pitch and roll of the \mbox{\hyperlink{classCMPS12}{CMPS12}}.
